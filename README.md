# Generative-AI

This repository contains the code I am practicing while reading *Hands-On Generative AI with Transformers and Diffusion Models*.

I created this project as a personal learning space to try out examples from the book, understand how the code works, and keep track of my progress with generative AI experiments.

## What This Repo Includes

- `generating_text/` for text-generation and NLP-related examples
- `generating_images/` for image-generation examples
- small practice scripts based on concepts from the book

## Purpose

The goal of this repository is to:

- practice the code examples from the book
- learn how Hugging Face Transformers pipelines work
- experiment with text and image generation
- keep my learning progress organized in one place

## Environment

This project is being tested in a Python environment using libraries such as:

- `transformers`
- `torch`
- `diffusers`

Some examples may use GPU acceleration when CUDA is available.

## Notes

- This is a learning repository, so the code may change often as I test new ideas.
- Some scripts are simple experiments written while following along with the book.
- Outputs and generated files may differ depending on the installed models, package versions, and hardware setup.
